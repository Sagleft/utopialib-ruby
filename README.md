## Install

```bash
gem install utopialib
```

## How to run tests from examples

First, you need gem to load the environment variables, which is used in the examples:

```bash
gem install dotenv
```

Make changes to `.env`:

```bash
cp example.env .env
```

and run in cli:

```bash
ruby example2.rb
```
